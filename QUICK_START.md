# 🚀 COLOCAR ONLINE EM 2 MINUTOS - GUIA RÁPIDO

## ⚡ Opção MAIS FÁCIL: Railway.app (Recomendado)

### Passo 1: Aceda a Railway
👉 **https://railway.app**

### Passo 2: Login com GitHub
1. Clique em **"Login with GitHub"**
2. Autorize Railway

### Passo 3: Deploy Automático
1. Clique em **"+ New"** → **"Project"**
2. Selecione **"Deploy from GitHub repo"**
3. Escolha **`car-parking-multiplayer`**
4. Clique em **"Deploy"**

### Passo 4: Aguarde (2-3 minutos)
- Railway faz build automático
- Você receberá um **URL único**

### ✅ PRONTO! Seu jogo está ONLINE!

**Seu link será algo como:**
```
https://car-parking-multiplayer-prod.railway.app
```

---

## 🎮 Como Jogar

1. Abra o link que recebeu
2. Digite seu **nome**
3. Escolha seu **carro**
4. **Clique em "Nova Partida"** ou "Entrar em Jogo"
5. **Use WASD ou Setas** para mover
6. **Estacione no espaço cinzento** para ganhar pontos!

---

## 📊 Controles do Jogo

| Tecla | Ação |
|-------|------|
| **↑** ou **W** | Mover para frente |
| **↓** ou **S** | Mover para trás |
| **←** ou **A** | Virar à esquerda |
| **→** ou **D** | Virar à direita |
| **ESPAÇO** | Freio (mais slow) |
| **ESC** | Voltar ao menu |

## ⚠️ Se der Erro

### Erro: "Cannot find module"
```bash
npm install
git add package-lock.json
git commit -m "fix: update lock file"
git push
```

### Erro: "WebSocket connection failed"
- Aguarde 2-3 minutos para Railway iniciar completamente
- Recarregue a página (F5)

### Erro: "Port already in use"
- Railway gerencia portas automaticamente
- Não precisa fazer nada!
