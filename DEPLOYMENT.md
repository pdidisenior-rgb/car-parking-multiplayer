# 🚀 Guia de Deployment\n\n## Opção 1: Railway.app (RECOMENDADO - Mais Fácil)\n\n### Passo 1: Aceda ao Railway\n1. Vá para https://railway.app\n2. Clique em \"Start a New Project\"\n3. Selecione \"Deploy from GitHub repo\"\n\n### Passo 2: Conecte GitHub\n1. Autorize Railway a acessar suas repos\n2. Selecione `car-parking-multiplayer`\n3. Clique em \"Deploy\"\n\n### Passo 3: Configure variáveis\n- Não precisa de configuração adicional!\n- Railway detecta `railway.json` automaticamente\n\n### Passo 4: Aguarde Deploy\n- Railway fará build e deploy automático\n- Você receberá um URL único\n\n✅ **Pronto! Seu jogo estará online em segundos!**\n\n---\n\n## Opção 2: Heroku\n\n### Passo 1: Instale Heroku CLI\n```bash\n# Windows/macOS/Linux\nchoco install heroku-cli  # ou apt-get install heroku-cli\n```\n\n### Passo 2: Login\n```bash\nheroku login\n```\n\n### Passo 3: Create App\n```bash\ncd car-parking-multiplayer\nheroku create seu-app-name\n```\n\n### Passo 4: Deploy\n```bash\ngit push heroku main\n```\n\n### Passo 5: Abra\n```bash\nheroku open\n```\n\n✅ **Seu jogo está em: https://seu-app-name.herokuapp.com**\n\n---\n\n## Opção 3: Vercel + Backend em Railway\n\n### Frontend (Vercel)\n1. Crie `vercel.json`:\n```json\n{\n  \"buildCommand\": \"echo 'Frontend ready'\",\n  \"outputDirectory\": \"client\"\n}\n```\n\n2. Deploy:\n```bash\nnpm i -g vercel\nvercel\n```\n\n### Backend (Railway)\n- Siga a Opção 1 apenas para `/server`\n\n---\n\n## Monitoramento\n\n### Railway\n- Dashboard automático em https://railway.app\n- Logs em tempo real\n- Métricas de uso\n\n### Heroku\n```bash\nheroku logs --tail\nheroku ps:scale web=1\n```\n\n---\n\n## Troubleshooting\n\n### Erro: \"Cannot find module\"\n```bash\nnpm install\ngit add package-lock.json\ngit commit -m \"fix: add lock file\"\n```\n\n### Porta não disponível\n- Railway/Heroku usam PORT automático\n- Server detecta via `process.env.PORT`\n\n### Conexão WebSocket falha\n- Verifique CORS em `server/index.js`\n- Use origem absoluta, não localhost\n\n---\n\n## Variáveis de Ambiente\n\nAdicione no Railway/Heroku se necessário:\n```\nPORT=5000\nNODE_ENV=production\n```\n\n---\n\n## Custo\n\n- **Railway:** Gratuito até 5GB/mês\n- **Heroku:** Plano gratuito descontinuado (pago a partir de $7/mês)\n- **Vercel:** Gratuito para frontend estático\n\n🎉 **Recomendação: Use Railway.app - é grátis, fácil e rápido!**\n
