# Guia de Contribuição

Obrigado por seu interesse em contribuir para o Car Parking Multiplayer!

## Como Contribuir

### 1. Fork o Repositório
```bash
git clone https://github.com/pdidisenior-rgb/car-parking-multiplayer.git
cd car-parking-multiplayer
```

### 2. Crie uma Branch
```bash
git checkout -b feature/sua-feature
```

### 3. Faça suas Mudanças
- Mantenha o código limpo e bem documentado
- Siga o estilo de código existente
- Adicione comentários quando necessário

### 4. Commit
```bash
git commit -m "feat: descrição da sua feature"
```

### 5. Push e Pull Request
```bash
git push origin feature/sua-feature
```

## Padrões de Código

### JavaScript
- Use camelCase para variáveis e funções
- Use PascalCase para classes
- Use const por padrão, let quando necessário
- Evite var

### Commits
- feat: nova feature
- fix: correção de bug
- docs: documentação
- style: formatação
- refactor: refatoração
- test: testes
