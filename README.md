# 🚗 Car Parking Multiplayer - Versão Melhorada

Um jogo de estacionamento multiplayer em tempo real com física realista, múltiplos mapas e competição online!

## ✨ Features Principais

- 🎮 **Multiplayer em Tempo Real** - Joga com até 4 jogadores simultaneamente
- 🗺️ **Múltiplos Mapas** - 5 garagens diferentes com layouts únicos
- 🚗 **Variedade de Carros** - 8 modelos diferentes com características únicas
- ⭐ **Sistema de Pontuação** - Ranking global e estatísticas
- 📊 **Física Realista** - Direção e estacionamento desafiador
- 🎯 **Modo de Desafios** - Missões com objetivos específicos
- 💾 **Progressão de Jogador** - Save de dados e histórico

## 🛠️ Tech Stack

- **Frontend:** HTML5 + Phaser 3 (Game Engine)
- **Backend:** Node.js + Express + Socket.io
- **Database:** MongoDB (opcional)
- **Deployment:** Heroku/Railway + GitHub Pages

## 📁 Estrutura do Projeto

```
car-parking-multiplayer/
├── server/              # Backend Node.js
├── client/             # Frontend Phaser
├── assets/             # Sprites, sons, tilesets
└── docs/              # Documentação
```

## 🚀 Começar

### Pré-requisitos
- Node.js v16+
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/pdidisenior-rgb/car-parking-multiplayer.git
cd car-parking-multiplayer

# Instale dependências
npm install

# Inicie o servidor
npm run server

# Em outro terminal, inicie o cliente
npm run dev
```

O jogo estará disponível em `http://localhost:3000`

## 🎮 Como Jogar

1. **Entrar no Jogo** - Escolha seu nick e carro
2. **Selecione o Mapa** - 5 garagens disponíveis
3. **Estacione Corretamente** - Ganhe pontos por precisão
4. **Suba no Ranking** - Compita com outros jogadores

## 🎯 Objetivos do Jogo

- Estacionar nos espaços designados
- Evitar colisões com outros carros
- Completar dentro do tempo limite
- Ganhar o máximo de pontos possível

## 📈 Pontuação

- **Estacionamento Perfeito:** 100 pontos
- **Bom Estacionamento:** 80 pontos
- **Aceitável:** 50 pontos
- **Bonus Tempo:** +10 pontos por segundo restante
