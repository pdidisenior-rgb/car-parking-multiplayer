## 🚀 Como Começar AGORA

### Opção 1: Railway.app (RECOMENDADO)
1. Vá para https://railway.app
2. Clique "Login with GitHub"
3. Clique "+ New Project"
4. Selecione "Deploy from GitHub"
5. Escolha seu repositório
6. Pronto! O jogo está ONLINE em 3 minutos! 🎉

### Opção 2: Local
```bash
git clone https://github.com/pdidisenior-rgb/car-parking-multiplayer.git
cd car-parking-multiplayer
npm install
npm run dev
```
Aceda a: `http://localhost:5000`
